Admin sign in

Log in with a GitHub token to create or edit posts.

Needs a token with write access to this repo's contents.

How to create one: GitHub → Settings → Developer settings → Fine-grained tokens → New token → set Repository access to just this repo → under Permissions, set Contents: Read and write → Generate.

The token is stored only in this browser (if you check "remember"), never sent anywhere except directly to GitHub's API. Anyone with the token can edit this repo, so keep it private and revoke it from GitHub's settings any time.
