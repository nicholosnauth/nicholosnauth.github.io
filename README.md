# Portfolio Website (Minimal Scaffold)

This repository now contains a minimal, responsive single-page portfolio scaffold.

Files updated:

- `index.html` — new single-page layout (hero, about, projects, contact)
- `main.css` — modern responsive styles
- `script.js` — small vanilla JS for nav, smooth scroll, year and simple form validation

To preview locally:

```bash
# from repository root
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```

If you need a specific design, assets, or want to preserve legacy jQuery features (sliders, parallax, magnify), tell me which parts to restore and I'll integrate them selectively.
